- **Create a project**:

```sh
pnpm create vite@latest
```

- **Add storybook**:

```sh
pnpm create storybook@latest
```

- **Add storybook addons**:

```sh
pnpm install -D @storybook/addon-interactions @storybook/addon-links msw msw-storybook-addon
```
